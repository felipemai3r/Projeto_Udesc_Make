## Sobre o Projeto

Construa um robô seguidor de linha do zero usando Arduino e sensores infravermelhos. Este projeto é ideal para quem quer aprender eletrônica e programação de forma prática e divertida.

## Materiais Necessários

- Arduino Uno R3
- Sensores infravermelhos (x2)
- Driver motor L298N
- Motores DC (x2)
- Chassi de acrílico
- Rodas (x2) + rodabolos
- Jumpers e protoboard
- Bateria 9v

## Passo a Passo

### Passo 1: Montar o chassi
Fixe os motores DC no chassi de acrílico usando parafusos. Instale as rodas e a rodabolsa na parte traseira.

### Passo 2: Conectar o driver de motor
Ligue o driver L298N aos motores DC e conecte a alimentação da bateria 9V.

### Passo 3: Instalar os sensores
Posicione os sensores infravermelhos na parte frontal do robô, apontados para o chão.

### Passo 4: Programar o Arduino
Carregue o código que lê os sensores e controla os motores para seguir a linha preta.

### Passo 5: Testar e calibrar
Coloque o robô sobre uma pista com fita isolante preta e ajuste a sensibilidade dos sensores.

## Resultado Final

Ao final deste projeto você terá um robô funcional capaz de seguir uma linha preta de forma autônoma, pronto para competições e demonstrações.
