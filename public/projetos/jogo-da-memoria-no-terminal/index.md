## Sobre o Projeto
Projeto simples em Python que cria um jogo da memória no terminal. Ensina estruturas de repetição, listas e lógica condicional.

## Passo a Passo
### Passo 1: Criar estrutura do jogo
Defina a lista de cartas e embaralhe os valores.


### Passo 2: Criar lógica de jogada
Permita que o usuário escolha duas posições.


### Passo 3: Validar acertos
Compare os valores e informe se houve acerto.


### Passo 4: Finalizar jogo
Encerre o jogo quando todos os pares forem encontrados.
